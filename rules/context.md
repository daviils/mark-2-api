# CONTEXTO

## FRAMEWORK
este é um projeto em Nest na versão 11.0.0 que usa TypeORM para criar tabelas automaticamente e é configurado com graphQL

## MODULOS
normalmente cada modulo possui 4 pastas

### DTO
pasta onde será criado os arquivos de modelo de input e alguns output

### ENTITY
é onde ficará os arquivos que será usado para criar as tabelas

### SERVICE
é  onde ficará os arquivos de construção de querys

### RESOLVER
é onde ficará os arquivos de construção do graphql

### RESUMO
caso eu pessa para construir um modulo sempre crie nessa estrura e caso precise de uma base use o modulo src/user

## VIEWS
dentro dessa pasta é onde ficará meus aquivos de views, é usuado a lib ejs

## CONCLUSÃO
espero usar vc para automação de desenvolvimento de modules e criação de views, e outros ajustes no geral
